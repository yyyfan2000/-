# 分镜头脚本

## 片头
- 时长：20 秒
- 画面：深色背景 + 标题动画“协同办公应用”
- 文案关键词：少点几步、少等几次、少切几个系统
- 画面建议：用关键词依次浮现，最后落到 App 首页 mockup

## 镜头 1：首页与产品定位
- 时长：25 秒
- 画面：录屏进入首页，展示“今日待办”和五大快捷入口
- 旁白重点：移动优先、五大高频场景、面向员工/管理者/行政/HR
- 屏幕字幕：会议预定 / 流程审批 / 差旅出行 / 文档查询 / 请假打卡

## 镜头 2：会议预定
- 时长：40 秒
- 画面：进入会议模块，展示“立即可用的会议室”卡片
- 操作：点击 A303 的“立即预订”，再展示会议详情页
- 强调点：30 秒内完成高频动作；按距离、容量、设备智能推荐
- 补充镜头：展示签到提醒或会议延长入口

## 镜头 3：流程审批
- 时长：40 秒
- 画面：进入审批页，展示“我发起的 / 待我审批”切换
- 操作：点开一条年假申请，展示审批信息和状态
- 强调点：少填表、自动流转、进度可见、审批决策有依据
- 屏幕字卡：发起快 / 进度清 / 决策准

## 镜头 4：差旅出行
- 时长：40 秒
- 画面：展示出差申请或出差提醒场景
- 操作：查看出差目的地、推荐交通和住宿、进入报销/出差状态联动
- 强调点：推荐方案、自动打通申请到报销链路
- 屏幕字卡：智能推荐 / 成本预估 / 自动联动

## 镜头 5：文档查询
- 时长：30 秒
- 画面：进入文档模块，搜索“差旅政策”
- 操作：打开《差旅政策 V2.0》，再展示相关文档推荐
- 强调点：搜得到、版本准、有关联
- 屏幕字卡：最新版 / 已生效 / 相关推荐

## 镜头 6：请假打卡
- 时长：30 秒
- 画面：展示到公司自动提醒打卡、忘打卡补卡入口
- 操作：切换模拟状态，出现打卡浮动按钮或补卡按钮
- 强调点：智能提醒、异常处理、统一状态中心
- 屏幕字卡：到点提醒 / 一键打卡 / 自动补救

## 镜头 7：结尾收束
- 时长：20 秒
- 画面：首页总览 + 五大功能快速回闪
- 结束语：让协同办公更少步骤，更高效率
- 结尾字幕：协同办公应用｜让高频协同动作打开即用
