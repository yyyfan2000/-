# 协同办公应用

> 让协同办公「少点几步、少等几次、少切几个系统」—— 一款移动优先的智能协同办公应用

## 项目简介

协同办公应用是一款面向中大型企业的移动优先协同办公平台，覆盖**会议预定、差旅出行、流程审批、文档查询、请假打卡**五大核心场景。通过智能化推荐与自动化流程，帮助员工用手机快速完成高频协同动作，同时让管理者与行政降低资源成本与管理成本。

## 五大核心场景

| 场景 | 核心能力 |
|------|----------|
| 🏢 **会议预定** | 一键秒订空闲会议室、智能推荐最优会议室、参会签到提醒 |
| ✈️ **差旅出行** | 智能推荐交通住宿方案、审批通过后一站式订票、出差结束一键生成报销单 |
| 📋 **流程审批** | 请假/出差/报销/采购审批、关键信息前置辅助决策、待办集中管理 |
| 📄 **文档查询** | 智能搜索与分类、文档版本管理、AI 摘要生成与上下文关联推荐 |
| 🕐 **请假打卡** | 智能打卡提醒（GPS/Wi-Fi/蓝牙）、出差自动打卡、忘打卡补卡提醒 |

## 快速开始

### 环境要求

- **Node.js** >= 18

### 安装与运行

```bash
# 进入应用目录
cd 协同办公应用

# 安装依赖
npm install

# 配置 Gemini API Key
# 复制 .env.example 为 .env.local 并填入你的 GEMINI_API_KEY

# 启动开发服务器
npm run dev
```

访问 `http://cowork-kappa-lyart.vercel.app` 即可查看应用。(需科学上网)

### 构建生产版本

```bash
npm run build
npm run preview
```

## 设计理念

- **移动优先**：所有核心功能优先为移动端设计，确保单手操作流畅
- **少步骤**：高频动作压缩到 3 步以内完成
- **智能预填**：表单能自动填的就自动填，减少用户输入
- **实时反馈**：用户操作后立即给出加载状态、成功/失败提示
- **AI 驱动**：借助 Gemini AI 实现智能搜索、文档摘要、上下文推荐
